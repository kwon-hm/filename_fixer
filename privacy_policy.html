개인정보처리방침

최종 수정일: 2024년 11월 14일

📋 핵심 요약

이 확장프로그램은 사용자의 개인정보를 수집하거나 전송하지 않습니다.

모든 처리는 사용자의 컴퓨터에서 로컬로 이루어지며, 외부 서버로 데이터가 전송되지 않습니다.

1. 수집하는 개인정보

수집하지 않습니다.

이 확장프로그램은 다음과 같은 개인정보를 수집하지 않습니다:

  • 이름, 이메일 주소, 전화번호 등 개인 식별 정보
  • 파일 내용
  • 파일명 (처리 후 즉시 삭제)
  • 사용 기록 또는 분석 데이터
  • 위치 정보
  • 기타 개인정보

2. 개인정보의 처리 목적

이 확장프로그램은 개인정보를 수집하지 않으므로 처리 목적이 없습니다.

모든 파일 처리 작업은 사용자의 브라우저 내에서만 이루어지며, 외부로 전송되지 않습니다.

3. 개인정보의 처리 및 보유기간

처리하지 않습니다.

파일명 변환 작업은 사용자의 컴퓨터에서 실시간으로 이루어지며, 처리된 파일명은 다운로드 후 즉시 메모리에서 삭제됩니다.

서버에 저장되거나 보관되는 데이터가 없습니다.

4. 개인정보의 제3자 제공

제공하지 않습니다.

개인정보를 수집하지 않으므로 제3자에게 제공할 정보가 없습니다.

5. 개인정보처리 위탁

위탁하지 않습니다.

모든 처리는 사용자의 컴퓨터에서 로컬로 이루어지며, 외부 서비스에 위탁하지 않습니다.

6. 정보주체의 권리·의무 및 행사방법

이 확장프로그램은 개인정보를 수집하지 않으므로, 정보주체의 권리 행사가 필요하지 않습니다.

사용자는 언제든지 확장프로그램을 삭제할 수 있으며, 삭제 시 모든 관련 데이터가 제거됩니다.

7. 개인정보의 파기

파기할 데이터가 없습니다.

개인정보를 수집하지 않으므로 파기할 데이터가 없습니다.

파일명 변환 작업은 메모리에서만 이루어지며, 작업 완료 후 즉시 삭제됩니다.

8. 개인정보 보호책임자

이 확장프로그램은 개인 프로젝트로 개발되었으며, 개인정보를 수집하지 않습니다.

문의사항이 있으시면 GitHub 이슈를 통해 연락해 주세요.

9. 개인정보 처리방침 변경

이 개인정보처리방침은 법령, 정책 또는 보안기술의 변경에 따라 내용의 추가, 삭제 및 수정이 있을 시에는 변경사항의 시행일 7일 전부터 이 페이지를 통해 공지하겠습니다.

10. 사용하는 권한

이 확장프로그램은 다음 권한을 사용합니다:

  • downloads: 복원된 파일을 사용자의 다운로드 폴더에 저장하기 위해 필요합니다.

이 권한은 파일 다운로드에만 사용되며, 개인정보 수집이나 전송에는 사용되지 않습니다.

11. 데이터 수집 및 전송

✅ 데이터 수집 없음

  • 외부 서버로 데이터를 전송하지 않습니다
  • 사용자 행동을 추적하지 않습니다
  • 분석 도구를 사용하지 않습니다
  • 쿠키를 설정하지 않습니다
  • 로컬 스토리지에 개인정보를 저장하지 않습니다

12. 보안

이 확장프로그램은 사용자의 파일을 안전하게 처리합니다:

  • 모든 파일 처리는 사용자의 브라우저 내에서만 이루어집니다
  • 파일 내용은 읽지 않으며, 파일명만 처리합니다
  • 네트워크 연결 없이 작동합니다
  • 오프라인에서도 사용 가능합니다

이 개인정보처리방침은 2024년 11월 14일부터 적용됩니다.
